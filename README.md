#  Keepy — безопасный мессенджер для родителей и детей

**Keepy** — это веб-мессенджер, разработанный для безопасного общения между детьми и взрослыми.  
Основная цель проекта — дать родителям контроль над цифровым окружением ребёнка и защитить его от нежелательного контента.

---

##  О проекте

Keepy создаётся как удобная и безопасная платформа для общения, где:
- родители могут управлять аккаунтами детей
- дети могут общаться в защищённой среде
- система автоматически фильтрует нежелательный контент

Проект разрабатывается в рамках учебной командной работы.

---

##  Основной функционал

###  Для родителей:
- Регистрация и авторизация
- Создание чатов и бесед
- Отправка текстовых сообщений
- Отправка медиа и аудио файлов
- Создание детских профилей
- Управление аккаунтами детей

###  Для детей:
- Общение в чатах
- Автоматическая фильтрация сообщений
- блокировка нежелательных слов

---

##  Особенности проекта

-  Безопасная среда общения для детей  
-  Родительский контроль  
-  Фильтрация нежелательного контента

##  Анализ мессенджеров

| Критерий | Telegram | WhatsApp | MAX (Госуслуги) |
|----------|----------|----------|-----------------|
| Аудитория в РФ | ~85 млн | ~90 млн | ~5 млн (растёт) |
| Шифрование | Секретные чаты (E2E) | Все личные чаты (E2E) | Не заявлено |
| Макс. размер файла | 2 ГБ (4 ГБ Premium) | 2 ГБ | 2 ГБ |
| Видеозвонки | Да, до 1000 участников | Да, до 32 участников | Да, групповые |
| Боты и автоматизация | Полноценная платформа | Ограниченно | Базовые функции |
| Каналы/Сообщества | Да, неограниченно | Только группы | Нет |
| Синхронизация устройств | Облачная, мгновенная | Через телефон | Облачная |
| Серверы в РФ | Нет | Нет | Да |
| Соответствие 152-ФЗ | Частично | Частично | Полное |
| Риск блокировки | Средний | Средний | Минимальный |
| Платные функции | Telegram Premium | Бизнес-тарифы | Бесплатно |
| Платформы | Все + Desktop | Все + Desktop | iOS, Android, Web |
| Интеграции | Боты, Mini Apps, API | Business API | Госуслуги, госорганы |
