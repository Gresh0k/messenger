# Keepy Messenger (v7)

Keepy — учебный веб-мессенджер на Flask с личными и групповыми чатами, вложениями и интерфейсом в темной цветовой гамме.

## Что есть в текущей версии

- Регистрация и вход пользователей
- Личные диалоги (1 на 1)
- Групповые чаты (создание бесед и обмен сообщениями)
- Отправка текста и файлов (изображения, видео, документы)
- Эмодзи-панель в окне чата
- Избранные чаты
- Контакты и быстрый старт диалога из контактов
- Статусы доставки/прочтения сообщений
- Real-time обновление сообщений через Socket.IO

## Технологии

- Python 3.10+
- Flask
- Flask-SocketIO
- SQLite
- bcrypt
- HTML, CSS, JavaScript

## Быстрый запуск

1. Перейдите в папку проекта:

```powershell
cd "D:\УЧЕБА\6_Semester\Thalyaev\Версии\7 версия"
```

2. Создайте и активируйте виртуальное окружение:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

3. Установите зависимости:

```powershell
pip install -r requirements.txt
```

4. Запустите приложение:

```powershell
python app.py
```

5. Откройте в браузере:

- [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Создание группового чата

1. Откройте страницу `Чаты`.
2. В блоке `Новая группа` укажите:
   - название группы;
   - участников через запятую (по username), например: `user1, user2`.
3. Нажмите `Создать группу`.

## Поддерживаемые типы вложений

Через `ALLOWED_EXTENSIONS` в `app.py`:

- Изображения: `png`, `jpg`, `jpeg`, `gif`
- Видео: `mp4`, `avi`, `mov`
- Документы и архивы: `pdf`, `doc`, `docx`, `zip`

## Примечания

- База данных `keepy_messenger.db` инициализируется автоматически при первом запуске.
- Папка `static/uploads` создается автоматически, если отсутствует.
- Проект учебный и не является production-ready решением.
